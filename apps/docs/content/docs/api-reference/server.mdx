## Routes

### route

Defines an upload route in your router.

```ts
route({
  // your route config...
});
```

## Request handling

### handleRequest

Handles a standard request object, can be used in any framework that uses standard Request and Response objects.

```ts
handleRequest(request, {
  // your router config...
});
```
